Keyboard Commands

Command Action
Tab / Shift + Tab Navigate between input slots, graphing viewport, and buttons.
Arrow Keys Move the graph cursor (crosshair) when focused on the graphing viewport.
Shift + Arrow Keys Move the graph cursor (crosshair) faster (coarser adjustment).
? Open this keyboard commands overlay.
H Hear Graph (automated 2.5s sweep left-to-right).
Alt + T Toggle Manual Keyboard Trace Mode.
Alt + G Draw Tangent Line (opens Tangent solver overlay directly).
Shift + S Zoom Standard (Resets viewport bounds to -10 to 10 for both axes).
Shift + F Zoom Fit (Fits the Y bounds to make all curves visible).
T Toggle between Graph View and Data Table View.
C Toggle Numerical Solvers Overlay Menu.
Alt + M Toggle Angle Mode (Radian vs. Degree).
Alt + P Cycle Decimal Precision (Float, Fix 2, Fix 3, Fix 4).
Alt + E Toggle Spreadsheet List Editor.
Alt + D Open Probability Distributions Menu (invNorm).
Escape Close open help or solver overlays / Exit spreadsheet.
Enter Force refresh and redraw the graphs.
