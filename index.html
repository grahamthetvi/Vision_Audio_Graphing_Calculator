Welcome

A talking graphing calculator built for low-vision and blind users.

Your Privacy

Everything runs entirely on your device. No data is ever sent anywhere. There are no servers, no accounts, no tracking, and no internet calls carrying your information. Your equations and settings stay in your browser.

Do you use a screen reader?

If you use a screen reader such as NVDA, JAWS, VoiceOver, or Orca, we will stay quiet and let your screen reader do the talking, so you do not hear everything twice.

If not, the built-in voice will speak responses aloud as you work.

You can change this any time in the Accessibility settings.

Keyboard Commands

Command Action
Tab / Shift + Tab Navigate between input slots, graphing viewport, and buttons.
Arrow Keys Move the graph cursor (crosshair) when focused on the graphing viewport.
Shift + Arrow Keys Move the graph cursor (crosshair) faster (coarser adjustment).
? Open this keyboard commands overlay.
H Hear Graph (automated 2.5s sweep left-to-right). Works when the graphing viewport is focused.
Alt + T Toggle Manual Keyboard Trace Mode.
Alt + G Draw Tangent Line (opens Tangent solver overlay directly).
Shift + S Zoom Standard (Resets viewport bounds to -10 to 10 for both axes).
Shift + F Zoom Fit (Fits the Y bounds to make all curves visible).
G Switch to Graph View.
T Toggle between Graph View and Data Table View.
C Toggle Numerical Solvers Overlay Menu.
Alt + M Toggle Angle Mode (Radian vs. Degree).
Alt + Shift + H Toggle between Graphing Workspace and Home Screen.
Alt + R Toggle Math Mode (Real vs. a+bi Complex).
Alt + Shift + P Cycle Decimal Precision (Float, Fix 2, Fix 3, Fix 4).
Alt + Shift + E Toggle Spreadsheet List Editor.
Alt + Shift + D Open Probability Distributions Menu (invNorm).
Alt + A Jump to Accessibility settings (voice, theme, font, letter weight and outline).
Escape Close open help or solver overlays / Exit spreadsheet.
Enter Force refresh and redraw the graphs.

Screen reader note: Some shortcuts use Alt + Shift combinations to avoid conflicts with browser menus and NVDA. Single-letter keys (G, T, C) only work when you are not typing in a text field. H (Hear Graph) requires the graphing viewport to be focused.
